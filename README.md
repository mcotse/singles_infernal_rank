# Singles Infernal Rank

A mobile-first PWA for ranking things with drag-and-drop. Features a hand-drawn aesthetic.

## Features

- Create ranking boards for anything
- Drag-and-drop to reorder rankings
- Add photos and notes to cards
- Hand-drawn visual style
- Works offline (PWA)
- Sample data: Singles Inferno Season 5 cast

## Tech Stack

- Vite + React 18 + TypeScript
- Tailwind CSS
- Framer Motion
- IndexedDB + localStorage

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173/singles_infernal_rank/

## Load Sample Data

Go to **Settings** and click **Load Cast & Photos** to load the Singles Inferno Season 5 cast as sample data.
